Este proyecto trata de englobar un desarrollo de una aplicación web en el mundo real, contando con:

Benvenida Login Roles Tokens
Bienvenida

Una página de bienvenida que sirve para que el usuario se sienta cómodo desde el instante que entra a la aplicación.

Login

Un sistema de login conectado a una cloud database, con autenticación y autorización.

Roles

Por medio de la autenticación se nos permitirá manejar distintas vistas para distintos roles de usuarios. En este caso serán usuario y administrador..

Tokens

Por medio de la autorización se permitirá acceder a distintas páginas.
